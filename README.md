# Read Plugin README

一个用于程序员在上班时偷偷阅读书籍的VSCode插件，让你在编辑代码的同时，在主侧栏阅读书籍。

## 功能特性

### 📚 核心功能
- **.txt文件阅读**：支持加载和阅读.txt格式的书籍文件
- **阅读进度缓存**：自动保存阅读进度，重启VSCode后自动定位到上次阅读位置
- **字体大小调节**：支持增大或缩小字体，适应不同阅读需求
- **多书籍管理**：支持添加多本书籍并在它们之间自由切换
- **界面美化**：简约美观的界面设计，支持暗色模式，保护眼睛

### 🔒 隐蔽性设计
- **集成于VSCode**：作为侧栏视图存在，与代码编辑完美融合
- **低调的图标**：使用书籍图标，不引人注目
- **自然的界面**：采用与VSCode风格一致的设计，看起来像普通的开发工具

### 🎨 用户体验
- **响应式设计**：适应不同屏幕尺寸
- **流畅的动画**：界面切换和状态变化有平滑过渡效果
- **直观的操作**：简单易用的操作界面，减少学习成本

## 安装方法

### 方法一：从VSCode marketplace安装
（未来将发布到 marketplace）

### 方法二：从源码安装
1. 克隆本仓库到本地
2. 打开VSCode并进入插件目录
3. 运行 `npm install` 安装依赖
4. 运行 `npm run compile` 编译插件
5. 按 `F5` 启动调试会话，或使用 `vsce package` 生成 `.vsix` 文件后安装

## 使用指南

### 添加书籍
1. 点击VSCode左侧活动栏中的「Read」图标打开插件
2. 在「Books」视图中，点击右键选择「Add Book」命令
3. 在文件选择对话框中选择一个 `.txt` 文件
4. 书籍将被添加到列表中，点击即可开始阅读

### 阅读操作
- **打开书籍**：在「Books」列表中点击书籍名称
- **切换书籍**：直接点击其他书籍即可切换
- **调节字体**：使用命令面板（`Ctrl+Shift+P`）运行「Increase Font Size」或「Decrease Font Size」命令
- **查看进度**：书籍名称旁边会显示当前阅读进度百分比

### 管理书籍
- **删除书籍**：在「Books」列表中右键点击书籍，选择「Remove Book」

## 技术实现

### 核心技术
- **VSCode Extension API**：使用官方提供的扩展API开发
- **TypeScript**：采用TypeScript编写，确保类型安全
- **本地存储**：使用VSCode的全局存储API保存书籍信息和阅读进度
- **Webview**：使用Webview技术渲染书籍内容，提供丰富的展示效果
